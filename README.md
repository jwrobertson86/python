# Jason W. Robertson's Repository

This is where some of my Python examples live as a demonstration of what I've learned over the years. More will be uploaded soon, but for now we have:
* `Blackjack` - A Blackjack game, demonstrating the use of classes, custom exceptions, and system timing functions, for a Udemy course in Python completed in 2019.
* `Kenken` - An independently devised project, created because I was tired of trying to figure out all the different possibilities for solving certain blocks in more complex [kenken puzzles](https://en.wikipedia.org/wiki/KenKen).
* `Simple Exercises` - A couple of other, simpler games made as part of a Udemy course mentioned above.

In the near future, I will be going through the rest of the coding projects on my hard drive and seeing what also makes sense to add. At minimum, I'll probably add a couple of the TensorFlow examples I created as part of a more recent TF 2 class I took.
